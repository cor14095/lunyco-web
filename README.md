# Lunyco Web


Modern portfolio built with **Next.js 15 (App Router, Server Actions)** and **Tailwind**.


## Structure
- `src/` - Source folder
- `src/app/` – Routes & RSC
- `src/content/` – Mock data
- `src/components/` – UI
- `src/server/` – Server Actions
- `src/interfaces/` - Project interfaces and types


## Development
```bash
npm ci
npm run dev
```


Deploy via AWS Amplify (SSR enabled).
